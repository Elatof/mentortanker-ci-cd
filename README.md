# build-tools
## 1. Maven
#### to build project
```bash
mvn build
```
#### to run test
```bash
mvn test
```
## 2. Gradle
#### to build project
```bash
gradle build
```
#### to run test
```bash
gradle test
```
## 3. ANT+Ivy
#### to build project
```bash
ant -buildfile default.build build
```
#### to clean project
```bash
ant -buildfile default.build clean
```
#### to run test
```bash
ant -buildfile default.build test
```
